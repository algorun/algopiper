AlgoPiper Documentation
========================

AlgoPiper is a web-based software tool that offers a graphical user interface to create, run, integrate and share pipelines consisting of algorithms packaged with `AlgoPiper <http://AlgoPiper.org>`_. This page provides a tour through AlgoPiper interface and how to use it to create computational pipelines, run them on a given dataset, integrate them with other software tools using a RESTful API, and share them on AlgoPiper website. It also gives examples as cases of study for using the tool.
